# 📚 Sistema de Avaliação por Competências

Projeto desenvolvido em Java para automatizar o cálculo do desempenho acadêmico de estudantes em uma rede de escolas particulares.

## Funcionalidades

* Cadastro de 4 notas bimestrais
* Validação das notas (0 a 10)
* Cálculo da média ponderada
* Verificação da frequência escolar
* Classificação do aluno em:
  
  * Aprovado
  * Exame Final
  * Reprovado
* Cálculo da média após Exame Final

## Regras de Avaliação

* Média ≥ 7,0: Aprovado
* Média entre 5,0 e 6,9: Exame Final
* Média < 5,0 ou frequência < 75%: Reprovado

## Tecnologias

* Java
* JOptionPane

## Objetivo

Praticar conceitos de programação como validação de dados, estruturas condicionais, cálculos matemáticos e tomada de decisão.
